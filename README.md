# Task 3: CSS Units, Box Models, and Fonts

Ye project TuteDude ke Task 3 ke liye banaya gaya hai. Iska maqsad CSS Box Model, Viewport units, aur custom styling (Gradients) ko bina Flexbox ya Position properties ke implement karna hai.

## 🚀 Features & Implementation

- *HTML Structure*: Ek clean semantic HTML structure tayaar kiya gaya hai jisme image container div aur interactive button shamil hai.
- *Viewport Units*: Spacing ke liye vw (Viewport Width) aur vh (Viewport Height) ka istemal kiya gaya hai.
- *Layout Logic*: 
  - Screen ke left aur right se *25%* equal spacing di gayi hai.
  - Top aur bottom se *10%* spacing rakhi gayi hai.
- *No Scroll*: box-sizing: border-box ka sahi istemal kiya gaya hai taaki page par unnecessary scrolling na ho.
- *Gradient Button*: Button ko ek modern gradient background aur custom font settings di gayi hain.
- *Restrictions Followed*: 
  - ❌ No Flexbox used.
  - ❌ No position: relative or absolute used.
  - ❌ No overflow: hidden used.

## 📁 File Structure

- index.html - Main HTML document.
- style.css - External stylesheet jisme saari styling aur units define hain.
- Readme.md - Project documentation.

## 🛠️ How to Run

1. Saari files (index.html, style.css, Readme.md) ko ek hi folder mein rakhein.
2. index.html ko kisi bhi modern browser (Chrome, Edge, Firefox) mein open karein.
3. Page ko resize karke check karein, layout viewport ke hisaab se adjust hoga.
